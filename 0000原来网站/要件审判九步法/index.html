    要件审判九步法¶

    ==== 要件审判九步法 ====

    https://pic.baike.soso.com/ugc/baikepic2/11331/20160803015309-365092936.jpg/300

    === 一、要件审判的内容 ===

    《要件审判》一书是邹碧华法官于2010年11月1日在法律出版社出版的一本小册子，本书是实践与理论相结合，详细介绍了“要件审判九步法”的具体内容：

    *  第一步 固定权利请求；

*  第二步 确定权利请求基础规范；

*  第三步 确定抗辩权基础规范；

*  第四步 基础规范构成要件分析；

*  第五步 诉讼主张的检索；

*  第六步 争点整理；

*  第七步 要件事实证明；

*  第八步 事实认定；

*  第九步 要件归入/作出裁判。

    本书着眼于提高法官的逻辑思维能力和法律适用能力，对法官审理案件和制作裁判文书具有很好的指导意义。

    这本书的知识框架建立的前提就是使用者的社会角色的定位是居中裁判，而涉诉双方是对抗制。

    === 二、启发 ===

    而就纪委监委的工作而言，对我们又有什么启法呢？

    1、就工作结果而言，我们的业务部门的调查结论实际上是既要对举报人负责、又要向被举报人负责。

    2、就调查展开依据和涉及的内容而言，举报人的陈述与被举报人的陈述客观上具有一定的差异性，这是难以避免的事情。

    3、就机关职责定位而言，纪委监委机关的定位是政治机关，调查申请要合规，实施要得当，同时还要考虑到政治影响和社会影响等问题，并不是单纯的业务办案的问题。同法官的社会调解功能非常类似的是，纪委监委机关在某些情况下也是起到社会调解、化解问题、解决问题的调节剂作用。

    因些，我们可以看到纪检机关与审判机关在职责定位、工作内容、工作方式等等有着诸多的相似。

    === 三、影响 ===

    那么我们将要件审判九步法迁移过来是否行的通呢？

    第一步 固定权利请求；询问举报人有什么请求、要求？他是有否有权利被侵害？

    第二步 确定权利请求基础规范；看看举报人这些请求、要求有什么证据、材料？权利被侵害是否“到人、到物”，是否详尽详实，还是否空洞无实质？

    第三步 确定抗辩权基础规范；在识别出举报人的请求之后，我们要对被举报人的答辩、答复、回答进行审查。 ^ 种类 | 方式 | ^ 备注 ^ | 第一种 | 沉默 || 不置可否 | | 第二种 | 否认 | 消极的否认 | 我没干过贪污的事 | |:::|:::| 积极的否认 | 不是贪污，是单位欠我的钱，欠我的工资 | | 第三种 | 不知 || 我不知道有这回事 | | 第四种 | 抗辩 | 权利消灭抗辩 |不是我不公示信息，是别的单位公示，且已经公示过了，不存在他们没有知情权的问题 | |:::|:::| 权利妨碍抗辩 |不是我不公示信息，是要经过程序才能公示，要开党委会，我一个人说了不算 | | 第五种 | 自认 || 有这回事 |

    注：否认和抗辩的区别 p83

    第四步 基础规范构成要件分析；逻辑推理中大前提的确认

    一个法条往往是由构成要件和法律效果（后果）两部分组成，有两种结构：

    1、要件1+要件2+···=后果。

    不同的要件导致一个后果，比如：对于受到改组处理的党组织领导机构成员，除应当受到撤销党内职务以上（含撤销党内职务）处分的外，均自然免职。

    2、要件1或要件2···=后果。

    要件选择性适用导致一个后果，比如：党员受到警告处分一年内、受到严重警告处分一年半内，不得在党内提升职务和向党外组织推荐担任高于其原任职务的党外职务。

    第五步 诉讼主张的检索；对照法条，对涉案双方的观点梳理、思考，看看他们的观点证据有无遗漏要点。举报人举报的事要点是否全面，是否客观的表达出来了，被举报人的回答、抗辩是否全面，有无要点、关键证据被遗漏？ 甲：乙是贪污犯，证据123

    乙：我不是贪污犯，证据123

    法律规范：贪污3000元者，死。

    调查人员思考：要办这个案子，一要是贪污，二要是3000元，三才能死；甲说的贪污这个主张和证据是否能达到“贪污”这个构成要件的标准？乙的辩解是否达到、是否排除贪污、3000元这两个要件的标准?有无遗漏？

    第六步 争点整理；总结出关键法律构成要件，从法律、事实两个方面入手作为展开调查的重点。

    第七步 要件事实证明；需要查明的事实争点整理出来后，看案件事实是否充分，需要哪些证据、谁来提供证据。

    要查明事实，就要了解信息，要了解信息就要接触和掌握承载该信息的载体，这个过程就是调查、他证、自证的过程，在办案过程中要考虑到谁来提供证据的问题。

    需要说明的是，纪律检查部门所涉及到业务并没有法院、检察院、政府、公安等等部门的时效性问题，当后者未在时效提出诉求、提供证据，未遵守法律流程中的时间规则，就要面临最后的不利后果，但是证据的内容、整体等等都会受到时间的影响发生变化，这是一个客观规律，所以纪检业务常常会面临一个“可能有这种情况，但无证据证实”的境况，这种“查无实据”的结果是我们只要向举报人申明的，且该申明过程在调查展开前是比较好的。

    取证受到时间的影响是无法避免的，我们要从整体上有个好的把握，大致分为五个层次（如下图）。 | | 是否已经产生 | 是否知道存在 | 是否占有控制 | 是否有取证的必要 | | 第一层次 ^ 新产生 | | | | | 第二层次 | 已经产生 | 尚不知道 | | | | 第三层次 | 已经产生 | 已经知道 | 尚未占有控制 | | | 第四层次 | 已经产生 | 已经知道 | 已经占有控制 | 尚无取证的必要 | | 第五层次 | 已经产生 | 已经知道 | 已经占有控制 | 具取证的必要 |

    因此，从一个调查部门办案来讲，就证据而言，核心证据还是非核心证据，必须取证的、非必须取证都是要考虑的东西。

    第八步 事实认定；

    1、事实基于证据

    我们所了解的事实如果无法通过证据来证实，这个待证事实的陈述是不具有足够的说服力，而作为纪检监察机关基于证据的事实认定才是我们开展业务的核心和方向。

    2、要从“三性”上进行判断 合法性 证据的形式、收集和运用符合法律的规定。不具有合法性就丧失了证据资格。一般通过形式合法性审查、程序合法性审查两个方面来进行审查。 关联性 证据事实与案件事实存在关联。有证明价值和意义的证据才可被纳入取证范畴之内。 *真实性和证明力的判断，这个是重点和核心，要考虑以下几个因素：①证据来源，②动机因素，提供证据的人与当事人有无利害关系，是影响证据内容、证明力的重要因素，③认知因素，当事人、证人的理解能力、记忆能力、年龄、心理、表达等等会影响证据的证明力，④证据产生的场所、过程、载体等等细节，⑤证据的内容与其他关系证据的是否存在协调性、冲突性问题。

    3、就证明力而言，足够的证明力就可以形成证据优势，信息的有效叠加可以消除待证事实、事物的不确定性，根据法律规范要件的每个证明点，要分为单个证据的认定和证据综合优势的判定两个层面的问题。 单个证据的认定，要解决的是证据的“三性”问题。 证据综合优势的判定，要解决的是证明力大小的问题，也就是是否可以消除不确定性，是否形成对于证据的优势判断。

    4、同法院审判部门相似的是，时过境迁是法庭的天然敌人，也是纪检机关调查部门天然的敌人。作为调查部门是不能在要件事实或者待证事实不明的情况下向审理、内查部门提交的，最关键的一个点是，我们是否穷尽了调查、证明途径，是否在举报人提供线索的可查性范围内最大程度的去发现案件事实。

    在时间这个维度下，调查取证的出发点和考虑因素可能就是从“谁主张，谁举证”、“谁掌握，谁举证、“谁优势，谁举证”三个方面进行，在待取证据的认知上，也是从确定知道有证据、不太确定有无证据、谁接近事实谁应当提供证据。 谁主张谁举证 解决的是事实的调查方向的问题 谁掌握谁举证 解决的是事实的证据来源的问题 *谁优势谁举证 解决的是事实的证明力的问题。

    比如甲告乙贪污，调查人员首先考虑的是“谁主张谁举证”，甲主张那么甲有何证据？下一步在进行调查中，就要考虑到贪污的钱款的“来源、走向”，“来源”的证据取自单位、关联单位的账目，“走向”取自银行、资产代持人的交易内容，随着调查的深入就可能涉及到一些无法掌握的内容，比如代理政府采购的招投标公司，政府机关与招投标公司之间应当有协议内容，但是政府机关提供的合同材料有瑕疵，无法完全反映出具体的全貌，可能只是一个“傀儡甲方”的存在，那么对于具体过程和内容，具体参与活动的“乙方人员和文件”就显现出证明力的差异性，乙方在证明过程和内容这个事实问题上就相对于“傀儡甲方”具有更强的证明力。

    第九步 要件归入 作出裁判）。
