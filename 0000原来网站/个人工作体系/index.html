    个人工作体系¶

    ===== 个人工作体系 =====

    每个人在工作中根据实际工作的需要慢慢形成的一套适合自己的工作方法、工具的集合。

    ==== 一、如何认识工作？ ====

    所有的组织系统都是通过一定的组织架构来完成对于人类个体组织起来完成、形成一定的功能。系统的好处在于通过一定的层级结构减少风险，缓解冲突，避免组织中的每个个体处于风险之中。系统都是一个典型的耗散结构，源源不断的输入信息、文档，再通过系统的加工、处理，输出一定的信息、文档。

    {{ :haosan.svg |}}

    ==== 二、方法论汇总 ====

    • 邮箱工作法 一般个人都有至少两个邮箱，工作邮箱和安全邮箱，可以作为个人日志的收集工具使用，这个工具用来记个人日志是比较好的，和网盘、日历等等系统的整合度一般都不错，就内容而言具有一定的客观性，一旦写完就无法修改，缺点是不是每个人都习惯使用邮箱来办公。 -纸质工作系统 传统的使用纸质笔记的方式来办公，市面上日本人所撰写的书籍大多提倡这类工具的使用，就国人而言，更多的是手机APP的使用，因为国内的网络系统环境是比较好的。 -电子工作系统

    ==== 三、时间管理 ====

    关于时间管理的书籍、视频和软件都是非常多的，就本质而言，时间管理不是一个非常贴切的名字，时间选择这个词是比较好的，时间管理不是要把你24个小时的时间弄成48个小时，而是在24个小时中分出哪些时间是要干啥的，哪些重要的事情需要快速去完成，哪些事情是要慢慢完成的。

    ==== 四、笔记工具 ====

    • anytype 个人现在主力在用的笔记软件，本地工具，同时打开自动同步笔记的内容，没有表格功能，和其他的笔记软件不同，用起来收集、汇总材料文件还是比较好的。 -notion 一款号称“all in one”的笔记软件，可以快速扩展和分享为个人网页。 -typora 本地markdown文件工具，可以用来摘录网页内容，比较简洁，非常喜欢。 -腾讯文档 就在线文档而言，钉钉、飞书等等也都是不错的，但是因为个人经常使用微信摘录一些网页上的内容，所以使用腾讯文档也是比较方便。 -思维导图工具 思维导图工具在日常生活和工作中的主要用途是梳理大脑里的思路，比如你看到一个案件之后所有的要素和关联，你把这个东西以思维导图的形式给展现出来，那样就可以起到一个非常好的效果，其中树形图/时间轴/鱼骨图都是比较常见的，大家可以自己去了解如何使用。 -白板工具 白板工具的主要用途是展示要素/拓展关联/形成思路，一般在公安机关办案过程中使用的是比较多的，白板工具相对于思维导图，白板更强调的是从已知到未知的拓展过程，思维导图更多的是决策工具。

    ==== 五、人际关系 ====

    现在是2022年6月份，在个人所接触的行业中，公安/法院/医院这几个行业都是属于需要大量处理人际冲突类的行业，这样的行业也是让人感觉到焦虑的行业。

    那么如何缓解行业焦虑,或者说是避免行业风险呢？

    心理学家阿德勒指出一切烦恼都是人际关系的烦恼，一大原因是没有把自己和他人的课题分清楚，要么干预别人的课题要么任由别人干预自己课题。只要能够学会课题分离，我们的心理或者人际关系就会发生巨大改变。

    比如别人对你提出要求，你的课题是判断要不要接受Ta的要求，只需要就事论事做出你想做的回应就好。至于Ta怎么来处理你的回应，他会不会感到失望，会不会认为你太不近人情，那是他的课题了。

    课题分离给我们提供了一个很好的思考角度，很多棘手的问题，用课题分离来判断，可能会让我们豁然开朗。

    但是，一件事来了，怎么来分辨这是谁的课题呢？阿德勒认为，只需要考虑一下“某种选择带来的结果最终要由谁来承担”就可以，谁来承担这个结果，那就是谁的课题，谁就有这件事的选择权和决定权。

    听起来很简单，但操作起来太难了！因为生活中，人跟人之间的牵绊，不是简简单单能够说得清楚的。尤其是在跟父母、配偶、孩子、朋友、老板、同事这些人的关系中，不仅要考虑这个事本身该怎么样，还要顾虑对方情感上的感受，并且承担对方这种感受带给你的影响。

    认清自己的课题，尽力完成它，尽力改善你跟外界的交互。当你做到这些之后，外界的评价如何，那就不是你的课题了。这时候你就可以坦然说出一句：他怎么看你，跟你无关。

    每一个行业都有自己的风险，短暂的接触一个行业，然后通过和行业深耕者沟通和交流，或者阅读一些比较典型性的案例，然后总结和归纳出所在行业的风险，而事实上，知道和做到之间是存在一道比较深的沟壑。不存在说哪一个行业是高枕无忧的，一个是随机应变，一个是但行好事莫问前程。
