    зҺӢжҷ“дёңжҠўеҠ«гҖҒжөҒж°“гҖҒзӣ—зӘғжЎҲиў«е‘ҠдәәпјҡзҺӢжҷ“дёңпјҢз”·пјҢ22еІҒпјҢеҢ—дә¬еёӮдәәгҖӮйҰ–йғҪй’ўй“Ғе…¬еҸёзү№ж®Ҡй’ўе…¬еҸёи–„жқҝеҺӮе·ҘдәәгҖӮ1996е№ҙ5жңҲ26ж—Ҙиў«йҖ®жҚ•гҖӮиҫ©жҠӨдәәпјҡе®Ӣеҫ·е®¶пјҢеҢ—дә¬еёӮз»ҸжөҺеҫӢеёҲдәӢеҠЎжүҖеҫӢеёҲгҖӮеҢ—дә¬еёӮдәәж°‘жЈҖеҜҹйҷўеҲҶйҷўд»Ҙиў«е‘ҠдәәзҺӢжҷ“дёңзҠҜжҠўеҠ«зҪӘгҖҒжөҒж°“зҪӘгҖҒзӣ—зӘғзҪӘеҗ‘еҢ—дә¬еёӮз¬¬дёҖдёӯзә§дәәж°‘жі•йҷўжҸҗиө·е…¬иҜүгҖӮеҢ—дә¬еёӮз¬¬дёҖдёӯзә§дәәж°‘жі•йҷўз»Ҹе®ЎзҗҶжҹҘжҳҺпјҡиў«е‘ҠдәәзҺӢжҷ“дёңдјҷеҗҢеә·е»әе№іпјҲеҸҰжЎҲеӨ„зҗҶпјүдәҺ1996е№ҙ1жңҲ23ж—ҘдёӢеҚҲ5ж—¶и®ёпјҢеҲ°жң¬еёӮзҹіжҷҜеұұеҢәеҸӨеҹҺз¬¬дәҢе°ҸеӯҰеҚ—е…¬е…ұз”өиҜқдәӯжү“з”өиҜқпјҢеә·е»әе№іејәиЎҢе°ҶжӯЈеңЁжү“з”өиҜқзҡ„жӣ№жӯЈйЈһпјҲз”·пјҢ30еІҒпјүзҡ„з”өиҜқжҢӮж–ӯгҖӮйҡҸеҗҺпјҢзҺӢжҷ“дёңгҖҒеә·е»әе№іеҸҲеҜ№жӣ№жӯЈйЈһиҝӣиЎҢж®ҙжү“пјҢжҠўеҠ«е…¶жүҖеёҰжҲ’жҢҮпјҢеӣ жӣ№жӯЈйЈһеҸҚжҠ—жҠўеҠ«жңӘйҖһгҖӮеҪ“жҷҡ6ж—¶и®ёпјҢзҺӢжҷ“дёңзӯүдәәеҸҲеңЁзҹіжҷҜеұұеҢәе…«и§’еҚ—и·Ҝ14ж ӢжҘјеҢ—дҫ§пјҢеҜ»жңәжҠўеҠ«дҪңжЎҲпјҢйҖӮйҖўеҗ•Г—пјҲеҘіпјүи·ҜиҝҮжӯӨеӨ„пјҢзҺӢжҷ“дёңдёҠеүҚжҠўиө°еҗ•зҡ„иғҢеҢ…гҖӮеҪ“еҗ•Г—иҝҪиө¶зҙўиҰҒж—¶пјҢзҺӢжҷ“дёңжҠ“дҪҸеҗ•Г—зҡ„еӨҙеҸ‘иҝӣиЎҢж®ҙжү“пјҢе°Ҷе…¶ж‘”еҖ’еңЁең°пјҢжҠўиө°йҮ‘йЎ№й“ҫдёҖжқЎгҖҒдәәж°‘еёҒ380е…ғпјҢе№¶еҜ№еҗ•Г—иҝӣиЎҢзҢҘдәөгҖӮеҪ“жҷҡ11ж—¶пјҢзҺӢжҷ“дёңеҸҲеҲ°зҹіжҷҜеұұеҢәе…«и§’еҚ—и·Ҝ10ж Ӣ6й—Ё93еҸ·пјҢз ҙзӘ—е…Ҙе®Өзӣ—зӘғзҺӢе°‘дёңзҡ„дәәж°‘еёҒ3.3дёҮдҪҷе…ғгҖҒ700дҪҷзҫҺе…ғеҸҠйҮ‘йЎ№й“ҫгҖҒйҮ‘жҲ’жҢҮзӯүзү©е“ҒпјҢжүҖзӘғй’ұгҖҒзү©е…ұд»·еҖјдәәж°‘еёҒ3.9дёҮдҪҷе…ғгҖӮиў«е‘ҠдәәзҺӢжҷ“дёңдәҺ1996е№ҙ1жңҲ27ж—ҘеҮҢжҷЁ1ж—¶и®ёпјҢеҲ°зҹіжҷҜеұұеҢәйӣ•еЎ‘е…¬еӣӯйҷ„иҝ‘еҜ»жңәжҠўеҠ«пјҢе°ҶеҘійқ’е№ҙеј Г—Г—гҖҒеҲҳГ—Г—пјҲеқҮзі»жІіеҚ—зңҒе®ңйҳіеҺҝдәәпјүжҲӘдҪҸпјҢз”ЁеҸҢжүӢжҠ“дҪҸ2дәәеӨҙеҸ‘пјҢйҖјиҝ«дәӨеҮәй’ұиҙўпјҢ2дәәиҜҙжІЎжңүпјҢзҺӢжҷ“дёңејәиЎҢд»Һеј Г—Г—иә«дёҠжҗңеҮәдәәж°‘еёҒ120дҪҷе…ғгҖӮе°”еҗҺпјҢзҺӢжҷ“дёңеҸҲеҜ№2дәәжӢіи„ҡзӣёеҠ иҝӣиЎҢж®ҙжү“пјҢе№¶еҜ№еј Г—Г—иҝӣиЎҢзҢҘдәөгҖӮиў«е‘ҠдәәзҺӢжҷ“дёңдәҺ1996е№ҙ5жңҲ14ж—Ҙжҷҡ10ж—¶и®ёпјҢеңЁжң¬еёӮзҹіжҷҜеұұеҢәе…«и§’еҚ—и·Ҝ14ж ӢжҘјеҢ—дҫ§иҠұеӣӯеӨ„еҜ»жңәжҠўеҠ«гҖӮе°ҶиҝҮи·ҜиЎҢдәәйғқГ—пјҲеҘіпјҢ29еІҒпјүжҲӘдҪҸпјҢжҠ“дҪҸе…¶зҡ„еӨҙеҸ‘пјҢжӢіеҮ»йқўйғЁпјҢе°ҶйғқГ—жү“жҳҸпјҢжҠўиө°йҮ‘йЎ№й“ҫдёҖжқЎгҖҒйҮ‘жҲ’жҢҮдёҖжһҡпјҢе№¶еҜ№йғқГ—иҝӣиЎҢзҢҘдәөгҖӮйғқГ—иў«ж®ҙжү“иҮҙи„‘йңҮиҚЎгҖҒйј»йӘЁйӘЁжҠҳгҖӮзҺӢжҷ“дёңдҪңжЎҲеҗҺиў«еҪ“еңәжҠ“иҺ·гҖӮдёҠиҝ°дәӢе®һпјҢжңүеҸ—е®ідәәжӣ№жӯЈйЈһгҖҒеҗ•Г—Г—гҖҒеј Г—Г—гҖҒеҲҳГ—Г—гҖҒйғқГ—еҸҠдәӢдё»зҺӢе°‘дёңзҡ„йҷҲиҝ°пјҢзҺ°еңәеӢҳжҹҘз¬”еҪ•гҖҒеҢ»йҷўиҜҠж–ӯиҜҒжҳҺгҖҒжҹҘиҺ·зҡ„иў«жҠўгҖҒиў«зӣ—зү©е“Ғе’Ңиөғзү©дҪңд»·иҜҒжҳҺзӯүеңЁжЎҲиҜҒе®һпјӣиў«е‘ҠдәәзҺӢжҷ“дёңдҫӣиҝ°зҡ„дҪңжЎҲж—¶й—ҙгҖҒең°зӮ№гҖҒжүӢж®өзӯүжғ…иҠӮдёҺдёҠиҝ°иҜҒжҚ®зӣёз¬ҰгҖӮеҢ—дә¬еёӮз¬¬дёҖдёӯзә§дәәж°‘жі•йҷўи®ӨдёәпјҢиў«е‘ҠдәәзҺӢжҷ“дёңзӣ®ж— еӣҪжі•пјҢеңЁе…ЁеӣҪејҖеұ•дёҘеҺүжү“еҮ»дёҘйҮҚеҲ‘дәӢзҠҜзҪӘжҙ»еҠЁж–—дәүдёӯпјҢд»ҚиҝӣиЎҢзҠҜзҪӘжҙ»еҠЁпјҢзҠҜзҪӘж°”з„°еҡЈеј гҖӮзҺӢжҷ“дёңд»Ҙйқһжі•еҚ жңүдёәзӣ®зҡ„пјҢз”ЁжҡҙеҠӣжүӢж®өеӨҡж¬ЎжӢҰи·ҜжҠўеҠ«е…¬ж°‘иҙўзү©пјҢе№¶е…Ҙе®Өз§ҳеҜҶзӘғеҸ–е…¬ж°‘иҙўзү©пјҢж•°йўқзү№еҲ«е·ЁеӨ§пјҢдё”еҜ»иЎ…ж»ӢдәӢпјҢдҫ®иҫұеҰҮеҘіжғ…иҠӮжҒ¶еҠЈпјҢе…¶иЎҢдёәи§ҰзҠҜдәҶгҖҠдёӯеҚҺдәәж°‘е…ұе’ҢеӣҪеҲ‘жі•гҖӢз¬¬дёҖзҷҫдә”еҚҒжқЎз¬¬дәҢж¬ҫгҖҒз¬¬дёҖзҷҫдә”еҚҒдәҢжқЎгҖҒз¬¬дёҖзҷҫе…ӯеҚҒжқЎзҡ„и§„е®ҡпјҢеҲҶеҲ«жһ„жҲҗжҠўеҠ«зҪӘгҖҒзӣ—зӘғзҪӘе’ҢжөҒж°“зҪӘгҖӮжүҖзҠҜжҠўеҠ«зҪӘгҖҒзӣ—зӘғзҪӘжғ…иҠӮеқҮзү№еҲ«дёҘйҮҚпјҢзӨҫдјҡеҚұе®іжҖ§жһҒеӨ§пјҢеә”еҪ“дҫқжі•дёҘжғ©пјӣжүҖзҠҜжөҒж°“зҪӘжғ…иҠӮжҒ¶еҠЈпјҢеә”еҪ“дҫқз…§е…ЁеӣҪдәәеӨ§еёёе§”дјҡгҖҠе…ідәҺдёҘжғ©дёҘйҮҚеҚұе®ізӨҫдјҡжІ»е®үзҡ„зҠҜзҪӘеҲҶеӯҗзҡ„еҶіе®ҡгҖӢз¬¬дёҖжқЎз¬¬дёҖйЎ№зҡ„и§„е®ҡпјҢд»ҺдёҘжғ©еӨ„гҖӮзҺӢжҷ“дёңдёҖдәәзҠҜж•°зҪӘпјҢдҫқз…§еҲ‘жі•з¬¬е…ӯеҚҒеӣӣжқЎзҡ„и§„е®ҡпјҢеә”еҪ“ж•°зҪӘе№¶зҪҡгҖӮж №жҚ®е…¶зҠҜзҪӘжҖ§иҙЁе’Ңжғ…иҠӮпјҢдҫқз…§еҲ‘жі•з¬¬дә”еҚҒдёҖжқЎз¬¬дёҖж¬ҫгҖҒз¬¬дә”еҚҒдәҢжқЎе’Ңз¬¬дә”еҚҒдёүжқЎз¬¬дёҖж¬ҫзҡ„и§„е®ҡпјҢиҝҳеә”еҪ“зӣёеә”йҷ„еҠ еүҘеӨәж”ҝжІ»жқғеҲ©гҖӮзҺӢжҷ“дёңиҷҪеңЁзҫҒжҠјжңҹй—ҙеқҰзҷҪйғЁеҲҶзҪӘиЎҢпјҢдҪҶжүҖзҠҜзҪӘиЎҢжһҒдёәдёҘйҮҚпјҢе…¶иҰҒжұӮд»ҺиҪ»еӨ„зҪҡзҡ„зҗҶз”ұдёҚе……еҲҶпјҢдёҚдәҲйҮҮзәігҖӮжҚ®жӯӨпјҢиҜҘйҷўдәҺ1996е№ҙ6жңҲ1ж—ҘеҲӨеҶіеҰӮдёӢпјҡиў«е‘ҠдәәзҺӢжҷ“дёңзҠҜжҠўеҠ«зҪӘпјҢеҲӨеӨ„жӯ»еҲ‘пјҢеүҘеӨәж”ҝжІ»жқғеҲ©з»Ҳиә«пјӣзҠҜзӣ—зӘғзҪӘпјҢеҲӨеӨ„жңүжңҹеҫ’еҲ‘еҚҒдә”е№ҙпјҢеүҘеӨәж”ҝжІ»жқғеҲ©дёүе№ҙпјӣзҠҜжөҒж°“зҪӘпјҢеҲӨеӨ„жңүжңҹеҫ’еҲ‘еҚҒе№ҙпјҢеүҘеӨәж”ҝжІ»жқғеҲ©дәҢе№ҙпјӣеҶіе®ҡжү§иЎҢжӯ»еҲ‘пјҢеүҘеӨәж”ҝжІ»жқғеҲ©з»Ҳиә«гҖӮдёҖе®ЎеҲӨеҶіе®ЈеҲӨеҗҺпјҢиў«е‘ҠдәәзҺӢжҷ“дёңд»Ҙе…¶еқҰзҷҪдәӨеҫ…дәҶзҪӘиЎҢе’ҢжҸӯеҸ‘дәҶд»–дәәзҠҜзҪӘпјҢиҰҒжұӮд»ҺиҪ»еӨ„зҪҡдёәз”ұпјҢеҗ‘еҢ—дә¬еёӮй«ҳзә§дәәж°‘жі•йҷўжҸҗеҮәдёҠиҜүпјӣзҺӢжҷ“дёңзҡ„иҫ©жҠӨдәәи®Өдёәе…¶жөҒж°“иЎҢдёәжғ…иҠӮдёҖиҲ¬пјҢжүҖзҠҜжҠўеҠ«зҪӘжғ…иҠӮе№¶йқһзү№еҲ«дёҘйҮҚпјҢе»әи®®дәҲд»Ҙд»ҺиҪ»еӨ„зҪҡгҖӮеҢ—дә¬еёӮй«ҳзә§дәәж°‘жі•йҷўз»Ҹе®ЎзҗҶи®ӨдёәпјҢдёҠиҜүдәәзҺӢжҷ“дёңж— и§ҶеӣҪжі•пјҢзҠҜзҪӘж°”з„°еҡЈеј пјҢе…¶д»Ҙйқһжі•еҚ жңүдёәзӣ®зҡ„еӨҡж¬ЎжӢҰи·ҜжҠўеҠ«е…¬ж°‘иҙўзү©пјҢе№¶еңЁе®һж–ҪжҠўеҠ«е…¬ж°‘иҙўзү©иҝҮзЁӢдёӯдҪҝз”ЁжҡҙеҠӣиҮҙдјӨдәӢдё»иә«дҪ“пјҢжғ…иҠӮзү№еҲ«дёҘйҮҚпјӣеӨңе…Ҙж°‘е®…з§ҳеҜҶзӘғеҸ–е…¬ж°‘з§Ғдәәиҙўзү©пјҢж•°йўқзү№еҲ«е·ЁеӨ§пјҢжғ…иҠӮзү№еҲ«дёҘйҮҚпјӣдё”еҸҲе…¬з„¶и—җи§Ҷжі•зәӘе’ҢзӨҫдјҡе…¬еҫ·гҖҒеҜ»иЎ…ж»ӢдәӢпјҢдҫ®иҫұеҰҮеҘіпјҢжғ…иҠӮжһҒдёәжҒ¶еҠЈпјҢзҺӢжҷ“дёңзҡ„иЎҢдёәеҲҶеҲ«жһ„жҲҗжҠўеҠ«зҪӘгҖҒзӣ—зӘғзҪӘе’ҢжөҒж°“зҪӘпјҢеқҮеә”дҫқжі•дёҘжғ©гҖӮзҺӢжҷ“дёңиў«жҠ“иҺ·еҗҺпјҢиҷҪжңүеқҰзҷҪйғЁеҲҶзҪӘиЎҢе’ҢжҸӯеҸ‘д»–дәәдёҖиө·дёҖиҲ¬зҠҜзҪӘиЎҢдёәзҡ„жғ…иҠӮпјҢдҪҶдёҚжһ„жҲҗйҮҚеӨ§з«ӢеҠҹпјҢдёҚиғҪжҚ®жӯӨдәҲд»Ҙд»ҺиҪ»еӨ„зҪҡгҖӮзҺӢжҷ“дёңдёҠиҜүзҗҶз”ұдёҚиғҪжҲҗз«ӢпјҢеә”дәҲй©іеӣһгҖӮиҫ©жҠӨдәәжүҖжҸҗзҺӢжҷ“дёңзҡ„жөҒж°“иЎҢдёәжғ…иҠӮдёҖиҲ¬пјҢжүҖзҠҜжҠўеҠ«зҪӘе…¶жғ…иҠӮе№¶йқһзү№еҲ«дёҘйҮҚзҡ„иҫ©жҠӨж„Ҹи§ҒпјҢзјәд№ҸдәӢе®һж №жҚ®е’Ңжі•еҫӢдҫқжҚ®пјҢдёҚдәҲйҮҮзәігҖӮдёҖе®Ўжі•йҷўж №жҚ®зҺӢжҷ“дёңзҠҜзҪӘзҡ„дәӢе®һпјҢзҠҜзҪӘжҖ§иҙЁгҖҒжғ…иҠӮе’ҢеҜ№дәҺзӨҫдјҡзҡ„еҚұе®ізЁӢеәҰеҜ№е…¶жүҖдҪңзҡ„еҲӨеҶіпјҢе®ҡзҪӘе’ҢйҖӮз”Ёжі•еҫӢжӯЈзЎ®пјҢйҮҸеҲ‘йҖӮеҪ“пјҢе®ЎеҲӨзЁӢеәҸеҗҲжі•пјҢеә”дәҲз»ҙжҢҒгҖӮжҚ®жӯӨпјҢиҜҘйҷўдҫқз…§гҖҠдёӯеҚҺдәәж°‘е…ұе’ҢеӣҪеҲ‘дәӢиҜүи®јжі•гҖӢз¬¬дёҖзҷҫдёүеҚҒе…ӯжқЎз¬¬пјҲдёҖпјүйЎ№и§„е®ҡпјҢиЈҒе®ҡеҰӮдёӢпјҡй©іеӣһзҺӢжҷ“дёңзҡ„дёҠиҜүпјҢз»ҙжҢҒеҺҹеҲӨгҖӮж №жҚ®жңҖй«ҳдәәж°‘жі•йҷўе…ідәҺжҺҲжқғй«ҳзә§дәәж°‘жі•йҷўж ёеҮҶйғЁеҲҶжӯ»еҲ‘жЎҲд»¶зҡ„и§„е®ҡпјҢеҢ—дә¬еёӮй«ҳзә§дәәж°‘жі•йҷўж ёеҮҶдәҶдёҖе®Ўжі•йҷўеҜ№зҺӢжҷ“дёңд»ҘжҠўеҠ«зҪӘгҖҒзӣ—зӘғзҪӘгҖҒжөҒж°“зҪӘе№¶зҪҡпјҢеҶіе®ҡжү§иЎҢжӯ»еҲ‘пјҢеүҘеӨәж”ҝжІ»жқғеҲ©з»Ҳиә«зҡ„еҲ‘дәӢеҲӨеҶігҖӮ
