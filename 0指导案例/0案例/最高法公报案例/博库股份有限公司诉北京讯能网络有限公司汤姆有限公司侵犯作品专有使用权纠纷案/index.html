её®еҠ©зҷ»иҪҪдҪңе“Ғзҡ„зҪ‘з«ҷдј ж’ӯдәҶдҪңе“ҒпјҢеӣ жӯӨи®ҫй“ҫиҖ…е№¶йқһдҪңе“ҒдҪҝз”ЁдәәгҖӮеҺҹе‘ҠеҚҡеә“е…¬еҸёжҢҮжҺ§иў«е‘Ҡи®ҜиғҪе…¬еҸёгҖҒжұӨе§Ҷе…¬еҸёдҫөзҠҜдё“жңүдҪҝз”ЁжқғпјҢе…¶дё»иҰҒзҗҶз”ұжҳҜпјҡи®ҜиғҪе…¬еҸёж—ўз„¶дёҺд»Ҡж—Ҙи§ҶзӮ№зӯҫи®ўдәҶеҗҲдҪңеҚҸи®®е…ұеҗҢдёәжұӨе§ҶзҪ‘з«ҷжҸҗдҫӣзҪ‘з«ҷеҶ…е®№пјҢжҺҲжқғжұӨе§ҶзҪ‘з«ҷй“ҫжҺҘд»Ҡж—ҘдҪңе®¶зҪ‘пјҢе°ұеә”еҪ“еҜ№й“ҫжҺҘжқҘзҡ„зҪ‘йЎөеҶ…е®№иҙҹжңүдәӢе…Ҳе®ЎжҹҘзҡ„д№үеҠЎгҖӮд»–д»¬з–ҸдәҺеұҘиЎҢжӯӨйЎ№д№үеҠЎпјҢйҖ жҲҗдҪңе“Ғиў«йқһжі•дј ж’ӯпјҢдё»и§ӮдёҠеҜ№дҫөжқғзҡ„еҸ‘з”ҹеӯҳеңЁиҝҮй”ҷпјҢеӣ жӯӨеә”еҪ“жүҝжӢ…иҙЈд»»гҖӮзӣ®еүҚпјҢз”ұдәҺзҪ‘з«ҷд№Ӣй—ҙжҷ®йҒҚеӯҳеңЁй“ҫжҺҘе…ізі»пјҢжүҚдҪҝдә’иҒ”зҪ‘иғҪеӨҹеҝ«жҚ·ең°дј ж’ӯж•°йҮҸе·ЁеӨ§зҡ„еҗ„з§ҚдҝЎжҒҜгҖӮеҰӮжһңиҰҒжұӮи®ҫй“ҫиҖ…и®ҫзҪ®й“ҫжҺҘж—¶пјҢеҝ…йЎ»еҜ№й“ҫжҺҘжқҘзҡ„еҶ…е®№жүҝжӢ…дәӢе…Ҳе®ЎжҹҘзҡ„д№үеҠЎпјҢж— з–‘дјҡдҪҝй“ҫжҺҘзҡ„еҠҹиғҪеҸ—еҲ°йҳ»зўҚпјҢиҝҷеҜ№дәҺдҝғиҝӣдә’иҒ”зҪ‘дёҡзҡ„еҸ‘еұ•жҳҜдёҚеҲ©зҡ„гҖӮеҗҢж—¶д№ҹеә”зңӢеҲ°пјҢи®ҫй“ҫиҖ…жҳҜдёәдәҶеўһеҠ жң¬зҪ‘з«ҷзҡ„и®ҝй—®йҮҸпјҢеҠӣеӣҫиҺ·еҸ–жӣҙеӨ§зҡ„з»ҸжөҺеҲ©зӣҠжүҚи®ҫзҪ®й“ҫжҺҘгҖӮи®ҫй“ҫиҖ…еңЁиҺ·еҫ—еҲ©зӣҠзҡ„еҗҢж—¶пјҢеә”еҪ“жҢүз…§жқғеҲ©дёҺд№үеҠЎеҜ№зӯүзҡ„еҺҹеҲҷпјҢеҜ№и®ҫзҪ®й“ҫжҺҘжҳҜеҗҰдјҡеҰЁе®ід»–дәәиЎҢдҪҝжқғеҲ©еұҘиЎҢйҖӮеҪ“зҡ„жіЁж„Ҹд№үеҠЎгҖӮеңЁжҺҲжқғй“ҫжҺҘзҡ„жғ…еҶөдёӢпјҢи®ҫй“ҫиҖ…жҳҜеҗҰжүҝжӢ…иҙЈд»»пјҢеҸҜд»ҘеҲҶдёӨз§Қжғ…еҪўпјҡеҰӮжһңи®ҫй“ҫиҖ…жҳҺзҹҘй“ҫжҺҘзҡ„дҪңе“ҒеӯҳеңЁжқғеҲ©дёҠзҡ„з‘•з–өпјҢд»Қз„¶дәҲд»Ҙй“ҫжҺҘпјҢе…¶иЎҢдёәж— з–‘её®еҠ©дәҶдҫөжқғдәәдј ж’ӯпјҢжү©еӨ§дәҶдҫөжқғз»“жһңпјҢзҷ»иҪҪиҜҘдҪңе“Ғзҡ„зҪ‘з«ҷе’Ңи®ҫй“ҫиҖ…йғҪеә”еҪ“жүҝжӢ…дҫөжқғиҙЈд»»гҖӮеҰӮжһңи®ҫй“ҫиҖ…дәӢе…ҲдёҚзҹҘйҒ“й“ҫжҺҘжқҘзҡ„дҪңе“ҒеӯҳеңЁжқғеҲ©дёҠзҡ„з‘•з–өиҖҢдәҲд»Ҙй“ҫжҺҘпјҢе…¶дё»и§ӮдёҠе°ұжІЎжңүдҫөжқғзҡ„ж•…ж„ҸпјҢеҪ“з„¶ж— йңҖжүҝжӢ…ж°‘дәӢиҙЈд»»пјҢиҜҘиҙЈд»»еҸӘиғҪз”ұзҷ»иҪҪдҪңе“Ғзҡ„зҪ‘з«ҷжүҝжӢ…гҖӮиў«е‘Ҡи®ҜиғҪе…¬еҸёдёҺд»Ҡж—Ҙи§ҶзӮ№зӯҫи®ўзҡ„еҗҲдҪңеҗҲеҗҢдёӯпјҢеҸӘзәҰе®ҡеҗҲдҪңдёәжұӨе§ҶзҪ‘з«ҷи®ҫи®Ўж–ҮеӯҰйў‘йҒ“гҖҒеҲ¶дҪңжңүе…іж Ҹзӣ®еҶ…е®№е’ҢејҖеұ•зӣёе…іж–ҮеҢ–жҙ»еҠЁпјҢз”ұжұӨе§ҶзҪ‘з«ҷдёҺд»Ҡж—ҘдҪңе®¶зҪ‘й“ҫжҺҘд»ҘеҸ–еҫ—иҝҷдәӣж Ҹзӣ®еҶ…е®№пјҢжІЎжңүзәҰе®ҡж Ҹзӣ®дёӯдҪҝз”Ёе“ӘдәӣдҪңе“ҒпјҢеӣ жӯӨи®ҜиғҪе…¬еҸёгҖҒжұӨе§Ҷе…¬еҸёж— жі•еңЁи®ҫзҪ®й“ҫжҺҘеүҚе°ұзҹҘйҒ“иў«й“ҫжҺҘзҡ„дҪңе“ҒеӯҳеңЁжқғеҲ©з‘•з–өгҖӮи®ҜиғҪе…¬еҸёеңЁзӯҫи®ўеҗҲеҗҢж—¶пјҢжіЁж„ҸеҲ°д»Ҙй“ҫжҺҘзҡ„ж–№ејҸдј ж’ӯд»–дәәдҪңе“ҒпјҢеҸҜиғҪж¶үеҸҠеҲ°дҪңе“ҒжқғеҲ©дәәзҡ„жқғеҲ©дҝқжҠӨй—®йўҳпјҢжүҖд»ҘеңЁи®ўз«ӢеҗҲеҗҢж—¶еҚіеҗ‘еҜ№ж–№жҸҗеҮәдәҶжҳҺзЎ®зҡ„жқғеҲ©дҝқиҜҒиҰҒжұӮпјҢе·Із»Ҹе°ҪеҲ°йҖӮеҪ“жіЁж„Ҹзҡ„д№үеҠЎгҖӮи®ҜиғҪе…¬еҸёгҖҒжұӨе§Ҷе…¬еҸёдёҺзҷ»иҪҪиҜҘдҪңе“Ғзҡ„зҪ‘з«ҷд№Ӣй—ҙпјҢеҜ№дј ж’ӯдҫөзҠҜеҺҹе‘ҠеҚҡеә“е…¬еҸёдё“жңүдҪҝз”Ёжқғзҡ„дҪңе“ҒдёҚеӯҳеңЁе…ұеҗҢзҡ„ж•…ж„ҸгҖӮеҚҡеә“е…¬еҸёиө·иҜүеҗҺпјҢи®ҜиғҪе…¬еҸёгҖҒжұӨе§Ҷе…¬еҸёеҫ—зҹҘй“ҫжҺҘеҰЁе®ідәҶд»–дәәиЎҢдҪҝжқғеҲ©еҗҺпјҢе°ұеҸҠж—¶ең°ж–ӯејҖдәҶй“ҫжҺҘгҖӮеӣ жӯӨпјҢи®ҜиғҪе…¬еҸёгҖҒжұӨе§Ҷе…¬еҸёжІЎжңүе®һж–Ҫдҫөжқғзҡ„иЎҢдёәпјҢдёҚеә”жүҝжӢ…дҫөжқғзҡ„ж°‘дәӢиҙЈд»»гҖӮеҚҡеә“е…¬еҸёдё»еј и®ҜиғҪе…¬еҸёгҖҒжұӨе§Ҷе…¬еҸёдҫөжқғпјҢзҗҶз”ұдёҚиғҪжҲҗз«ӢпјҢе…¶иҜүи®јиҜ·жұӮдёҚдәҲж”ҜжҢҒгҖӮз»јдёҠжүҖиҝ°пјҢеҢ—дә¬еёӮз¬¬дәҢдёӯзә§дәәж°‘жі•йҷўдәҺ2001е№ҙ4жңҲ27ж—ҘеҲӨеҶіпјҡй©іеӣһеҺҹе‘ҠеҚҡеә“е…¬еҸёзҡ„иҜүи®јиҜ·жұӮгҖӮжЎҲд»¶еҸ—зҗҶиҙ№1706е…ғпјҢз”ұеҺҹе‘ҠеҚҡеә“е…¬еҸёиҙҹжӢ…гҖӮе®ЈеҲӨеҗҺпјҢеҸҢж–№еҪ“дәӢдәәеқҮжңӘжҸҗиө·дёҠиҜүпјҢдёҖе®ЎеҲӨеҶіеҸ‘з”ҹжі•еҫӢж•ҲеҠӣгҖӮ
